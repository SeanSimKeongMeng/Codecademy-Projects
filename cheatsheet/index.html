HTML Cheatsheet

HTML Tags

TAG NAME DESCRIPTION
<p> Paragraph Contains and displays a block of text that defines a paragraph.
<h1-h6> Headings Represents a text heading for a section of content, with <h1> being the highest level of heading and <h6> being the lowest.
<div> Division Represents a generic division of content. It has no semantic meaning, but will separate its contents from the rest of the document.
<span> Span Used for grouping related text or elements for styling and scripting.
<em> Emphasize Represents text which is emphasized. Browsers will show the enclosed text in italics, by default.
<strong> Strong Used to identify text that is very important, or urgent.
<br> Line Break Represents a break in text. It is used when text needs to span multiple lines rather than being in-line, such as in an address.
<ul> Unordered List Represents an unordered list of items.
<ol> Ordered List Represents an ordered list of items.
<li> List Item Represents a single item in a list of items. It and the other list items must be wrapped in an <ol>, <ul>, or <menu> tag.
<img> Image Displays an image on the web page.
<video> Video Represents an interface for adding video content to the page.

HTML Attributes

ATTRIBUTE NAME DESCRIPTION
id Unique Identity Used to specify a unique id for an HTML element, which cannot be shared across multiple elements within the same HTML document.
